内容

本表单共有4个大类：

  1. 个人信息
  2. 兴趣类
  3. 个人趋向类
  4. 性格类

个人信息

在第一项：个人信息，只有当：

  1. 对方满足你填写的要求
  2. 你满足对方填写的要求

才会进行之后的匹配，否则不会进行匹配。

表单的目的就是 实现 双份满足双方的要求

电子邮件地址

匹配结果会公布在网页中，为保护隐私，不建议填写QQ邮箱和名字缩写的邮箱。

学号

选择学号的后三位即可，以防止重复投递

兴趣类

  1. 兴趣类共有若干选项，至少填写一项，建议填写3-7项即可。

  2. 自己的信息可以和要求对方的信息不一致。

  3. 该项采取百分制，50%为周围人的平均兴趣水平，对则项越喜爱。

  4. 权重越大，则这项在计算分数中，占比越高。

  5. 禁止所有选项全填高分，否则认为是“不健康”的数据，不进行匹配。

个人趋向

  1. 个人趋向类选项共有若干个选项，至少填写一项，也可以全部填写。

  2. 自己的信息可以和要求对方的信息不一致。

  3. 该项采取百分制，50%为周围人的平均值。

  4. 权重越大，则这项在计算分数中，占比越高。

  5. 禁止所有选项全填同样的数据，否则认为是“不健康”的数据，不进行匹配。

性格类

  1. 性格类选项共有若干选项，至少填写一项，也可以全部填写。
  2. 自己的信息可以和要求对方的信息不一致。
  3. 该项采取百分制，50%为周围人的平均值。
  4. 权重越大，则这项在计算分数中，占比越高。
  5. 禁止所有选项全填同样的数据，否则认为是“不健康”的数据，不进行匹配。

匹配公式

双方每匹配一次，每个人都有一个得分：


$$
分数(百分制) = 100分 * \sum_{所有项的和}^{} 相对权重 * \left | 自己要求的数据 - 对方填写的数据 \right | * 函数 * 双方该项权重都不为0
$$
其中：

  1. 相对权重: 相对权重 = 该项的权重 / 所有项的权重之和​
  2. 自己要求的数据： 自己表单中的第三列
  3. 对方填写的数据： 对方表单中的第二列
  4. 函数：从图中可以看到，自己要求的数据 与 对方填写的数据 越接近，则得分越高。

双方该项权重都不为0： 该项中，任何一方的权重为0，则认为他对该项没有兴趣，则该项得分为0。

最终双方的匹配值为：


$$
匹配值 = (男生的分数 + 女生的分数) / 2 (百分制)
$$

根据匹配值大小，为每人匹配一个对象。

安全性

  1. 1、收集学历信息的表：只确认你的学历信息，和学号后三位(避免有人重复投递)，建议其他信息建议遮挡后，再上传
  2. 2、一台电脑只收集数值信息，另一台电脑只收集学历信息，另一台用MATLAB计算匹配值，技术有限，只能从物理上保护数据安全吧(狗头)

结果公布

如果《个人信息》这一项匹配成功，系统会根据最终匹配值的最大值，为每一个值匹配一个对象
约一个星期之后，会公布匹配结果，展示的信息有：自己邮箱 对方邮箱 最终匹配值.

注意事项

  1. 本表第一次成型，难免考虑不周，会根据同学反馈，更新完整、并增删表单内容
  2. 其他学校同学若有兴趣，也欢迎参与
  3. 由于匹配机制限制，如果男女生数量不对等，则会出现一个匹配多人的可能，介意勿填表
  4. 本表会收集所有填表人的信息，并会分析、公布表的整体信息，为之后的填表人作参考，介意勿填表
  5. 由于水平有限，中间难免有错误，介意勿填表
  6. 还有一些选项并没有在表单中展示，也欢迎大家在评论区提出 自己想添加的内容，表单大概一个星期更新一次吧。
